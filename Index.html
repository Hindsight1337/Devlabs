🎓 Azure DevTest Labs for Skolebruk

Sett opp et komplett lab-miljø for dine elever på under 15 minutter. Med automatisk kostnadskontroll, sikkerhet og VM-administrasjon.

Opptil 30 elever Sikker tilgang Kostnadskontroll Auto-shutdown
Deploy til Azure nå

🎯 Hovedfunksjoner

Kostnadskontroll

Automatisk budsjettgrenser per elev med varsler ved 50%, 80% og 100% av budsjett.

Sikkerhet

Rollebasert tilgang hvor elever kun ser sine egne VM-er og ressurser.

Auto-administrasjon

VM-er starter automatisk kl. 08:00 og slås av kl. 22:00 for å spare kostnader.

Brukerhåndtering

Automatisk opprettelse av studentkontoer med begrensede rettigheter.

🚀 Deploy til Azure

1-klikk deployment

Klikk knappen under for å starte deployment-prosessen i Azure Portal

Viktig: Du må være logget inn på Azure Portal med administrator-rettigheter før du klikker deploy-knappen.

📖 Steg-for-steg veiledning

1
Logg inn på Azure Portal

Gå til portal.azure.com og logg inn med din administrator-konto.

2
Klikk "Deploy to Azure"

Klikk på deploy-knappen over. Du vil bli sendt til Azure Portal sin deployment-side.

3
Fyll ut parametere

Fyll ut skjemaet med informasjon om din skole:

  • School Name: Navn på skolen (eks: "IKTAgder")
  • Cohort ID: Kull/semester (eks: "H2025")
  • Number of Students: Antall elever (1-30)
  • Daily Cost Limit: Daglig budsjett per elev
4
Start deployment

Klikk "Review + create" og deretter "Create". Deployment tar ca. 10-15 minutter.

5
Kjør post-deployment script

Last ned og kjør post-deployment PowerShell-scriptet for å konfigurere avanserte policyer.
